# Прогресс видео

Рабочий чеклист для ролика `Спор на 100к: таро против натальных карт`.

У каждого пункта есть номер. Можно писать коротко:

```text
44 готово
44 не готово
```

Номера можно менять, если структура сценария меняется. Внутри одной версии чеклиста номера должны идти по порядку без пропусков.
