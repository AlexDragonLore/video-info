# Прогресс видео

Рабочий чеклист для ролика `Спор на 100к: таро против натальных карт`.

Отмечать готовность так:

```text
- [ ] не готово
- [x] готово
```

## Общий прогресс

- [ ] Правила спора зафиксированы
- [ ] Дедлайн зафиксирован
- [ ] Бюджет на рекламу зафиксирован
- [ ] Способ подсчета выручки зафиксирован
- [ ] Сайт Сани готов к показу
- [ ] Сайт Ромы готов к показу
- [ ] Все ключевые сцены сняты
- [ ] Все экранные записи собраны
- [ ] Все графики и титры подготовлены
- [ ] Черновой монтаж собран
- [ ] Финальный монтаж готов

## Обязательные съемочные куски

### Саня

- [ ] Саня объясняет, почему выбрал таро

- [ ] Саня использует ИИ для генерации логики или текстов
- [ ] Саня показывает первую версию сайта
- [ ] Саня находит проблему в сайте
- [ ] Саня быстро исправляет проблему
- [ ] Саня проверяет оплату
- [ ] Саня реагирует на первые цифры

### Рома

- [ ] Рома объясняет, почему выбрал натальные карты
- [ ] Рома формулирует аудиторию
- [ ] Рома собирает оффер
- [ ] Рома проектирует платный результат
- [ ] Рома использует ИИ для качества интерпретации
- [ ] Рома показывает первую версию сайта
- [ ] Рома усиливает упаковку после обратной связи
- [ ] Рома реагирует на первые цифры

### Общие сцены

- [ ] Саня и Рома спорят на 100 000 рублей
