# Прогресс видео

Рабочий чеклист для ролика `Спор на 100к: таро против натальных карт`.

У каждого пункта есть постоянный номер. Можно писать коротко:

```text
44 готово
44 не готово
```

Отмечать готовность в файле так:

```text
пункт 44: не готово
пункт 44: готово
```

Номера не менять, даже если порядок сцен потом поменяется.
